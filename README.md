# spring-boot-microservice-k8s

## Requirement
- Minikube
- Docker

## For local deployemnt 
### Start minikube cluster 
``sh start-cluster.sh``

### Deploy 
``sh install-all.sh``

# Or

### Using Skaffold 
``skaffold dev``

### Open Swagger
``sh open-api-docs.sh``
Account , Order and Payment swagger docs will open in your browser

### For logs
``sh <service name>-log.sh`` 
Eg: ``sh order-log.sh``
